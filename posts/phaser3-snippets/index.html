1

sprite.setBounce(1, 1);

collide with world bounds

设置为 true, 则与世界发生碰撞

1

sprite.setCollideWorldBounds(true);

set gravity

1

sprite.setGravityY(200)

set collider

设置碰撞， 允许一个组和一个精灵相互碰撞

1

this.physics.add.collider(sprite, group);

set immovable

1

sprite.setImmovable();

}
